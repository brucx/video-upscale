# Models Directory

This directory is for ONNX model files. Models are not included in the repository.

## Download Models

Download from [purevision-service releases](https://github.com/brucx/purevision-service/releases):

- `2x_PureVision.onnx` - ESRGAN pixel-unshuffle, 2x upscale + denoise
- `2x_PureScale_fp16.onnx` - RealPLKSR pixelshuffle, 2x upscale

## Alternative Location

You can also place models in `~/purevision-service/models/` or set the `MODELS_DIR` environment variable.
